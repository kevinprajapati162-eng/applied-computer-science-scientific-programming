# Applied Computer Science - Scientific Programming Coursework

This repository contains my Applied Computer Science coursework on scientific programming. The assignment is organised into ten questions, each covering a different programming, mathematical, or data-processing task using Python.

## Project Overview
The coursework includes matrix operations, eigenvalue analysis, complex function visualisation, data handling, advanced plotting, vectorisation, equation-based computation, data processing, and FFT-based image analysis. Each question is stored in a separate folder for clarity and better organisation.

## Topics Covered
- Matrix manipulation and eigenvalue analysis
- Complex function visualisation
- Data handling and visualisation tools
- Matrix and polynomial operations
- Advanced plotting and code efficiency
- Vectorisation
- Working with equations
- Data processing
- FFT applied to images

## Technologies Used
- Python

- Scientific Programming Concepts
